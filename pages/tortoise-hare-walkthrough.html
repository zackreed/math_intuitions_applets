Key Concept: Average Rate of Change

The average rate of change over a time interval \(\Delta t\) is given by the slope of the secant line connecting two points on the distance-time graph:

\(\text{Average Rate} = \frac{\Delta \text{Distance}}{\Delta \text{Time}} = \frac{D(t + \Delta t) - D(t)}{\Delta t}\)

Average Rate of Change Calculator

The code below defines a function \(D(t)\) (distance) and calculates the average rate of change \(\frac{D(t_2) - D(t_1)}{t_2 - t_1}\).

from sympy import * t = Symbol('t') # Define the distance function (e.g., t^2 for accelerating) D = t**2 # Define the time interval [t1, t2] t1 = 0 t2 = 2 # Calculate positions pos1 = D.subs(t, t1) pos2 = D.subs(t, t2) # Calculate Average Rate of Change avg_rate = (pos2 - pos1) / (t2 - t1) print(f"Function: D(t) = {D}") print(f"Interval: [{t1}, {t2}]") print(f"Position at t={t1}: {pos1}") print(f"Position at t={t2}: {pos2}") print(f"Average Rate of Change: {avg_rate}")
