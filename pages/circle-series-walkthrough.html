Key Concept: Geometric Series

A geometric series is a sum where each term is a constant ratio times the previous term:

\(S = a + ar + ar^2 + ar^3 + \cdots\)

where \(a\) is the first term and \(r\) is the common ratio between consecutive terms.

Your Goal: Through the visualization and questions below, you'll discover whether infinite series can have finite sums, and if so, what determines the sum's value!

Question 3: Finding the Limit Formula

You've seen that \(1 + \frac{1}{2} + \frac{1}{4} + \cdots\) approaches 2. Let's discover the general pattern!

(a) For a series with \(a = 1\) and \(r = \frac{1}{2}\), the sum is 2. Try to express 2 in terms of \(a\) and \(r\).

Hint: Notice that \(2 = \frac{1}{1 - \frac{1}{2}} = \frac{a}{1-r}\)

(b) Test this formula: For \(a = 3\) and \(r = \frac{1}{3}\), what should the infinite sum be?

(c) Based on these examples, what is the general formula for the sum of a convergent geometric series?

\(S = \frac{a}{1-r}\) when \(|r| < 1\)
\(S = a \cdot r\)
\(S = \frac{a + r}{2}\)

Use Python with SymPy to calculate geometric series sums and verify convergence.

Geometric Series Calculator

The code below computes partial sums and the infinite sum (if it converges).

from sympy import * n = Symbol('n') # Define the geometric series parameters a = 1 # First term r = S(1)/2 # Common ratio (use S() for exact rational) # Compute partial sum formula: S_n = a(1 - r^n)/(1 - r) S_n = a * (1 - r**n) / (1 - r) print(f"Geometric Series: a = {a}, r = {r}") print(f"\nPartial Sum Formula:") print(f"S_n = {S_n}") # Evaluate specific partial sums for k in [1, 2, 3, 5, 10]: partial_sum = S_n.subs(n, k) print(f"S_{k} = {partial_sum} = {float(partial_sum):.6f}") # Infinite sum (if |r| < 1) if abs(r) < 1: S_inf = a / (1 - r) print(f"\nInfinite Sum (converges):") print(f"S = {S_inf} = {float(S_inf)}") else: print(f"\nSeries diverges (|r| >= 1)")
