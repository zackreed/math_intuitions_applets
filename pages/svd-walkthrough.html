Singular Value Decomposition (SVD)

Interactive Exploration with Vector Projections

Introduction

The Singular Value Decomposition (SVD) is a fundamental matrix factorization that decomposes any matrix \(A\) into three matrices:

\(A = U \Sigma V^T\)

where:

  • \(U\) contains the left singular vectors (eigenvectors of \(AA^T\))
  • \(\Sigma\) is a diagonal matrix of singular values
  • \(V\) contains the right singular vectors (eigenvectors of \(A^TA\))

Interactive Vector Projection Applet

Use this applet to visualize how data points project onto a direction vector. The projection lengths and distances are related to the SVD of the data matrix.

How to use the applet:

  1. Drag the blue vector (marked with an X) to change the projection direction
  2. Drag any data point (purple circles) to move them
  3. Toggle Projections to see/hide the projection vectors
  4. Toggle Distances to see/hide the perpendicular distances
  5. Watch how the Projection Square Sum and Distance Square Sum change

Key Concepts

Connection to the Applet

  • The first singular vector (direction of \(v_1\)) is the direction that maximizes the projection square sum
  • The first singular value \(\sigma_1\) is related to the square root of the maximum projection square sum
  • The distance square sum represents reconstruction error and is minimized when projecting onto principal components
  • Rotating the blue vector in the applet is like exploring different projection directions to find the optimal one
